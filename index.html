Installation

Run the dev installer found at /installers/php/dev_installer.php — this is a command-line script, so run it like so:

php /installers/php/dev_installer.php
Follow the prompts. If you have any questions, see the README in /installers/php.

Admin Interface

Login using the email and password from the dev installer.

Demos

We're working to include demo data as a standard part of the dev install. For now you'll need to hack on these a bit to get them working — start by creating elements in the Admin Interface.
